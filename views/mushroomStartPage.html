Mushrooming


Welcome to the mushrooming application!


The main functionalities in this application are:

  • Viewing different mushroom findings across Sweden
  • Finding the closest mushroom picking places of desired mushrooms and routes to these
  • Making light spacetime analysis of these findings
  • Possibility to insert, update and delete own mushroom findings

Mushroom info


Closest mushroom


This application functionality will find the closest desired mushroom specie based on your location. In order to use this functionality you have to assign the desired mushroom specie that you are looking for. When you start typing the mushroom name the textbox will guide you to find the correct one.




Shortest path to the closest mushroom


This application functionality will find the closest desired mushroom specie based on your location and calculates the shorthest route there using walkable routes. In order to use this functionality you have to assign the desired mushroom specie that you are looking for. When you start typing the mushroom name the textbox will quide you to find the correct one.
