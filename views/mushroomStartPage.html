Mushrooming


Welcome to the mushrooming application!


The main functionalities in this application are:

  • Viewing different mushroom findings across Sweden
  • Finding the closest mushroom picking places of desired mushrooms and routes to these
  • Making light spacetime analysis of these findings
  • Possibility to insert, update and delete own mushroom findings

Find mushrooms with tools


Find mushroom tab includes functionalities that can be used as tools to find the closest desired mushroom and shortest routes to these.

Closest mushroom


This application functionality will find the closest desired mushroom specie based on your or manually specified location. In order to use this functionality you have to do the following:

  1. Type the mushroom specie that you are looking for
  2. Select whether to use your or manually specified location
  3. In order to specify your own location manually, click the "Specify location" and click the map

Shortest path to the closest mushroom


This application functionality will find the closest desired mushroom specie based on your or manually specified location and calculates the shortest route there using walk, cycle and driving travel modes. In order to use this functionality you have to do the following:

  1. Type the mushroom specie that you are looking for
  2. Select whether the routing uses your or manually specified location
  3. In order to specify your own location manually, select "Manually specify the location" from the radiobox, click the "Specify location" and click the map
  4. Select your preferred tarvel mode
